Detailed Description

This base class serves as base from which other exceptions derive . The Exception class itself derives from std::exception in order to be throwable, but hides the methods provided by the standard exception class since it provides equivalent methods of its own.

The macro THROW_EXCEPTION(exception_class, ...) can be used to simplify exception throwing.
