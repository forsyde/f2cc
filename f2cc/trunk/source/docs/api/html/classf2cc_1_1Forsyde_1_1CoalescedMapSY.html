Detailed Description

This class implements a specialized process coalescedMapSY which is not part of the ForSyDe standard. It is used to replace coalesced MapSY processes into a single process which contains all function arguments of the processes which it replaces. Thus, executing a single CoalescedMapSY process produces the same result as executing a series of MapSY processes.
