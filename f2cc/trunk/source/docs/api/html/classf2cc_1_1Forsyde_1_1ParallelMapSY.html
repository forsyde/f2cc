Detailed Description

This class implements a specialized process parallelmapSY which is not part of the ForSyDe standard. Instead, it is used to replace data parallel sections consisting of a UnzipxSY process, a set of MapSY or CoalescedMapSY processes, followed by a ZipxSY process, with a single ParallelMapSY process as it entails the same semantic meaning.
