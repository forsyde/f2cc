Detailed Description

The Config class provides methods for accessing the program-related settings. The settings are usually given through the command-line and there is a special constructor for parsing the command-line into a Config object.

Member Enumeration Documentation

enum f2cc::Config::TargetPlatform

Denotes the target platform to which the ForSyDe model is to be synthesized.

Enumerator:
C 

Sequential C code, where no processes are executed in parallel.

CUDA 

Sequential C code annotaded with CUDA directives, where data parallel processes are executed in parallel.
