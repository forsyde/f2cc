f2cc Documentation

f2cc is a software synthesis tool developed as part of the ForSyDe (Formal System Design) project. The name stands for "ForSyDe-2-CUDA C" and enables models to be synthesized into either C code or CUDA C code, the latter which can be compiled for parallel execution on a CUDA-enabled NVIDIA graphics card. Currently, the tool can handle models which contain the following process types:

  • f2cc::ForSyDe::SY::comb
  • f2cc::ForSyDe::SY::ParallelMap
  • f2cc::ForSyDe::SY::unzipx
  • f2cc::ForSyDe::SY::zipx
  • f2cc::ForSyDe::SY::delay

The tool recognizes simple data parallel patterns in a model (an unzipx process, followed by a series of comb processes, which all connect to a zipx process) and generate correct CUDA code for such regions. Support for additional process types and data parallel model patterns will be added in the future.

f2cc was originally developed by Gabriel Hjort Blindell as part of his Master thesis at KTH (Royal Institute of Technology), Sweden. For more information about f2cc, the reader is advised to consult the thesis report.
