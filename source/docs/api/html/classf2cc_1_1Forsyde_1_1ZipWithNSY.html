virtual void f2cc::Forsyde::ZipWithNSY::checkFunction ( CFunction &  function,
size_t  num_in_ports 
) const throw (InvalidProcessException) [protected, virtual]

Performs a series of checks:

  • The function must have either equal number of input parameters as the process has in ports, or equal number of input parameters as the process has in ports + 1 (for the out port).
  • If the function has the same number of input parameters as the process has in ports, then the function must return data (i.e. have return data type other than void) which also is not an array.
  • If the function has the same number of input parameters as the process has in ports + 1 (for the out port), then the function must not return data (i.e. have return data type void).
  • If any input parameter is an array or pointer, it must also be declared const. If the function returns void, then the last input parameter is not considered.

functionFunction to check.
num_in_portsNumber of in ports to the process.
